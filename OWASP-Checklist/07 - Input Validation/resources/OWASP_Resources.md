# OWASP Input Validation Testing Resources  

Reference: OWASP Testing Guide v4.2  

## Official Tools  

### XSS Testing  
- [XSSer](https://xsser.03c8.net/)  
- [BeEF (Browser Exploitation Framework)](https://beefproject.com/)  
- [XSStrike](https://github.com/s0md3v/XSStrike)  
- [XSS Hunter](https://xsshunter.com/)  

### SQL Injection  
- [SQLmap](https://sqlmap.org/)  
- [jSQL Injection](https://github.com/ron190/jsql-injection)  
- [NoSQLMap](https://github.com/codingo/NoSQLMap)  
- [Havij](https://github.com/owasp/havij)  

### Command Injection  
- [Commix](https://github.com/commixproject/commix)  
- [WebSploit](https://github.com/websploit/websploit)  

### HTTP Parameter Testing  
- [OWASP ZAP](https://www.zaproxy.org/)  
- [Burp Suite](https://portswigger.net/burp)  
- [Param Miner (Burp Extension)](https://portswigger.net/bappstore/17d2949a985c4b7ca092728dba871943)  

### XML Testing  
- [XXE Injection Payload List](https://github.com/payloadbox/xxe-injection-payload-list)  
- [OWASP XML Injection Testing Guide](https://owasp.org/www-community/vulnerabilities/XML_External_Entity_(XXE)_Processing)

- [FindBugs](http://findbugs.sourceforge.net/)
